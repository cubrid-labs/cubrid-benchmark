## Next Release — v1.0 — Comprehensive Benchmarks

- Reproducible benchmark results across Python, TypeScript, and Go
- Extended workloads: connect/disconnect, prepared statements, batch insert, concurrent select
- CI-automated nightly benchmark runs
- P95/P99 latency tracking and RSS memory metrics
- Performance charts and cross-language comparison analysis
- Rust benchmarks after cubrid-rs stabilization

## Compatibility

- Python 3.10+, Node.js 18+, Go 1.21+
- CUBRID 11.2 and MySQL 8.0 (comparison target)
- Docker-based reproducible environment
